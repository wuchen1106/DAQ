 {\bfseries  Welcome to the world of \doxyref{Roody}{p.}{classRoody}. }\section{Introduction}\label{index_Introduction}
\doxyref{Roody}{p.}{classRoody} is {\tt ROOT} application to display histograms and other contents from ROOT and HBOOK files and from the online connection to the MIDAS and ROME online data analyzers. It is meant for display of {\itshape \/}.root files and {\itshape \/}.hbook files. If running this application in conjunction with either the {\tt Midas analyzer } or the {\tt ROME analyzer}, it will allow to visualize {\itshape online\/} data retrieved through a socket port.
